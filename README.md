# TRL Branding System – The Right Lifestyle

> **Brand Identity Guidelines for The Right Lifestyle AI Automation Company**

This repository contains all visual identity, design language, and brand standards for TRL across all digital touchpoints.

---

## 🎨 Brand Philosophy

**Positioning:** Modern AI systems company built for creators and entrepreneurs  
**Personality:** Premium, minimalist, execution-focused, transparent  
**Aesthetic:** Clean, cinematic, dark-themed, future-oriented  
**Mood:** Professional, approachable, innovative, trustworthy

---

## 🎯 Core Brand Message

**Tagline:** "Automate the routine. Accelerate your legacy."  
**Promise:** Buy back your time. Scale what matters.  
**Proof:** AI systems built for how you actually work.

## 🎨 Visual Identity

### Color Palette

| Purpose | Color | Hex | RGB |
|---------|-------|-----|-----|
| **Primary Background** | Deep Black | #0b0b0b | 11, 11, 11 |
| **Secondary Background** | Card Background | #151515 | 21, 21, 21 |
| **Text Primary** | Pure White | #FFFFFF | 255, 255, 255 |
| **Text Secondary** | Light Gray | #888888 | 136, 136, 136 |
| **Accent/Borders** | True Black | #000000 | 0, 0, 0 |
| **Success** | Green | #10B981 | 16, 185, 129 |
| **Warning** | Amber | #F59E0B | 245, 158, 11 |
| **Error** | Red | #EF4444 | 239, 68, 68 |

**Rationale:** Dark theme reduces eye strain, conveys premium positioning, and aligns with tech/AI aesthetic.

### Typography

**Primary Font Family:** Inter, -apple-system, BlinkMacSystemFont, 'Segoe UI', sans-serif  
**Fallback:** System sans-serif

#### Sizing Scale
| Element | Size | Weight | Line Height | Usage |
|---------|------|--------|-------------|-------|
| H1 (Hero) | 52px | 700 Bold | 1.2 | Page titles, main headlines |
| H2 (Section) | 36px | 700 Bold | 1.3 | Section titles, major headings |
| H3 (Subsection) | 24px | 600 Semi-Bold | 1.4 | Card titles, feature names |
| Body (Large) | 18px | 400 Regular | 1.6 | Featured copy, intro text |
| Body (Regular) | 16px | 400 Regular | 1.6 | Main content, paragraphs |
| Body (Small) | 14px | 400 Regular | 1.5 | Labels, metadata |
| Caption | 12px | 500 Medium | 1.4 | Timestamps, fine print |

### Logo & Wordmark

**Format:** Text-based "TRL"  
**Style:** Bold, uppercase, 3-letter monogram  
**Color:** White on dark backgrounds | Dark on light backgrounds  
**Minimum Size:** 24px (digital) | 0.5" (print)  
**Clear Space:** Minimum 20% of logo width on all sides

**Placement:**
- Primary: Top-left navigation
- Secondary: Page footers
- Accent: Social media profiles, email signatures

## 🎬 Visual Style

### Photography & Imagery
- **Style:** Cinematic, aspirational, authentic
- **Tone:** Real people doing real work (not stock photos)
- **Composition:** Wide, generous spacing; focus on clarity
- **Color Treatment:** Consistent with dark theme palette

### Icons
- **Library:** Feather Icons / Heroicons
- **Size:** 20px (navigation), 24px (content), 32px (hero)
- **Style:** Clean, minimal stroke weight (1.5–2px)
- **Color:** White on dark backgrounds, dark on light backgrounds

### Spacing & Layout
- **Base Unit:** 8px grid system
- **Padding:** 16px (small), 24px (medium), 32px (large)
- **Margins:** 24px (tight), 48px (standard), 64px (spacious)
- **Max Width:** 1200px (desktop), 100% (mobile)

## ✍️ Voice & Tone

### Core Values
- **Authentic** – We share our real process, not polished marketing speak
- **Direct** – We're clear and concise; no fluff or buzzwords
- **Empowering** – We focus on what our clients can achieve
- **Practical** – We talk in outcomes, not features

### Messaging Examples

| Scenario | ✅ Do Say | ❌ Don't Say |
|----------|-----------|-------------|
| Describing automation | "Reclaim 15 hours per week" | "Leverage cutting-edge AI synergy" |
| About complexity | "Simple to set up, powerful in action" | "Enterprise-grade solution" |
| About price | "$500–$3,000 depending on scope" | "Affordable pricing models" |
| About results | "One client went from 40 to 20 support tickets daily" | "10x your productivity" |

### Writing Guidelines
1. Use active voice ("We build systems" not "Systems are built")
2. Write for skimmers (headers, bullet points, short paragraphs)
3. Lead with benefits, follow with details
4. Use specific numbers over percentages
5. Be honest about limitations

## 🌐 Digital Applications

### Website
- Dark theme with white text
- Hero section with clear value prop
- Feature cards in grid layout
- Call-to-action buttons in accent color
- Responsive design (mobile-first)

### Email
- Header with TRL logo
- Consistent typography and spacing
- Clear hierarchy with H1, H2, body
- Single primary CTA per email
- Footer with social links

### Social Media
- Profile picture: TRL logo (1:1)
- Banner: Dark background, hero image, tagline (3:1)
- Posts: Mix of education, behind-the-scenes, and results
- Stories: Authentic, real-time content

### Presentations
- Dark template with white text
- TRL logo in corner (not intrusive)
- Consistent typography and spacing
- Visuals support (don't overpower) the message

## 🎨 Component Examples

### Buttons
```
Primary Button: White text on dark background (#0b0b0b)
Hover State: Slight opacity increase, cursor pointer
Secondary Button: Border only, dark text/border
Disabled State: 50% opacity
```

### Cards
```
Background: #151515 (card background)
Border: 1px solid #2d2d2d
Padding: 24px
Border Radius: 8px
Hover: Slight shadow lift, border highlight
```

### Links
- **Color:** Match surrounding text
- **Underline:** On hover only
- **Visited:** Slightly lighter shade

## 📱 Responsive Design

- **Mobile:** <640px – Full width, 16px padding
- **Tablet:** 640px–1024px – 2-column layouts, 24px padding
- **Desktop:** >1024px – 3+ columns, 48px padding

---

## ♿ Accessibility Standards

✅ **WCAG 2.1 AA Compliance**
- Minimum contrast ratio: 4.5:1 (text), 3:1 (graphics)
- Accessible color combinations (not relying on color alone)
- Semantic HTML structure
- ARIA labels where needed
- Keyboard navigable (all interactive elements)
