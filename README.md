# TRL Branding System

The Right Lifestyle visual and brand identity guidelines.

---

## Color Palette

| Color | Hex | Usage |
|-------|-----|-------|
| **Black** | #000000 | Navigation, borders |
| **Dark Background** | #0b0b0b | Primary background |
| **Card Background** | #151515 | Sections, cards |
| **White** | #FFFFFF | Text, accents |

---

## Typography

- **Primary Font Family**: Arial, sans-serif
- **Fallback**: System sans-serif

### Sizing
- **H1 (Hero)**: 52px
- **H2 (Section)**: 36px
- **H3 (Card)**: 24px
- **Body**: 16px
- **Accent**: 14px

---

## Logo

- **Format**: Text-based "TRL"
- **Style**: Bold, uppercase
- **Position**: Top-left navigation
- **Color**: White on black background
- **Usage**: Consistent across all projects

---

## Design Philosophy

- **Aesthetic**: Clean, minimal, dark-themed
- **Approach**: Structured and scalable
- **Focus**: Readability and professionalism
- **Mood**: Modern, technical, professional

---

## Application Guidelines

All TRL projects should:
1. Use the defined color palette
2. Maintain typography standards
3. Keep consistent spacing and padding
4. Follow the dark theme aesthetic
5. Apply hover states and transitions
